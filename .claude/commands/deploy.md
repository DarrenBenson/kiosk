/deploy [files] - Deploy files to webserver1

## Usage

```
/deploy api/bins.php       # Deploy single file
/deploy -a                 # Deploy all files
/deploy -b -a              # Backup then deploy all
/deploy -g -a              # Deploy to test GUID
```

## Instructions

Invoke `deploy-skill` with the provided arguments.
